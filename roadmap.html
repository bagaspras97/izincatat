Ringkasan Eksekutif

Izin Catat adalah bot WhatsApp untuk pencatatan keuangan pribadi yang dapat diakses langsung melalui WhatsApp tanpa perlu install aplikasi tambahan. Dokumen ini menjabarkan rencana pengembangan produk secara bertahap dari MVP yang sudah ada menuju platform asisten pribadi yang komprehensif.

Visi Produk: Menjadi asisten pribadi terlengkap via WhatsApp — catat keuangan, jadwal, tugas, dan apapun yang perlu diingat, semua dalam satu nomor.

Fase 1 — Penguatan Fondasi

Fokus pada stabilitas, pengalihan database ke provider yang lebih scalable, dan penerapan AI intent detection agar bot terasa lebih natural.

1.1 Migrasi Database: Neon → Supabase

Prioritas: Segera

Alasan migrasi:

  • Supabase menyediakan Auth, Storage, dan Realtime dalam satu platform
  • Lebih siap untuk fitur dual akun (pasangan) di fase berikutnya
  • Row Level Security (RLS) native untuk keamanan data per user

Langkah teknis:

  • Buat project Supabase (region Singapore)
  • Update schema.prisma dengan directUrl untuk pgBouncer
  • Export data dari Neon, import ke Supabase
  • Update environment variable di VPS dan Vercel

1.2 AI Intent Detection

Prioritas: Segera

Perubahan pengalaman user:

Sebelum (format kaku) Sesudah (natural)
catat keluar 35000 makan siang makan siang 35rb
catat keluar 5000 parkir habis 5k buat parkir
catat masuk 5000000 gaji gaji masuk 5 juta

Implementasi: Menggunakan Groq LLM (llama-3.3-70b) untuk klasifikasi intent sebelum routing ke handler. Biaya tambahan: Rp 0 (masih dalam free tier untuk <300 user aktif).

Fase 2 — Migrasi Provider WhatsApp

Beralih dari Baileys ke provider resmi agar bot tidak bergantung pada perangkat pribadi dan lebih stabil untuk production.

2.1 Perbandingan Provider

Provider Harga Pesan/Bulan Voice Note Rekomendasi
Baileys (sekarang) Gratis Unlimited ✓ HP pribadi ikut menerima pesan
Fonnte Super Rp 165.000/bulan 10.000 ✓ Paling populer, free trial tersedia
Wablas Enterprise Rp 139.000/bulan Unlimited ✓ Lebih murah jika bayar tahunan (Rp 76rb/bln)

2.2 Langkah Migrasi

  • Siapkan SIM card baru + nomor WA khusus bot
  • Daftar dan uji free trial (prioritas: test webhook voice note)
  • Buat Express/Fastify webhook server menggantikan Baileys listener
  • Refactor semua sock.sendMessage() ke fungsi kirimPesan() via REST API
  • Test end-to-end di branch feat/provider-wa
  • Deploy ke VPS dan set webhook URL di dashboard provider
Catatan: Setelah migrasi, bot berjalan 100% di server — tidak ada ketergantungan pada perangkat pribadi. Biaya tambahan Rp 139.000–165.000/bulan mulai aktif di fase ini.

Fase 3 — Super App: Personal Assistant

Memperluas fungsi bot dari pencatat keuangan menjadi asisten pribadi lengkap. Setiap modul ditambahkan secara bertahap tanpa mengganggu fitur yang sudah ada.

3.1 Modul Jadwal & Reminder

Prioritas: Segera setelah Fase 2

  • Simpan jadwal kegiatan: "besok jam 9 meeting klien"
  • Reminder otomatis H-1 dan hari-H via scheduler (sudah ada infrastrukturnya)
  • Tampilkan jadwal hari ini/minggu ini

3.2 Modul Todo

  • Buat dan kelola daftar tugas: "todo beli kado ulang tahun ibu"
  • Tandai selesai: "selesai beli kado"
  • Tampilkan todo yang belum selesai

3.3 Modul Catatan Bebas

  • Simpan informasi penting: "simpan nomor rekening BCA 1234567890"
  • Cari catatan: "cari catatan rekening"

3.4 Modul Habit Tracker

  • Catat kebiasaan harian: "sudah olahraga", "minum air 8 gelas"
  • Laporan konsistensi mingguan

Fase 4 — Monetisasi & Growth

4.1 Sistem Autentikasi

Prioritas: Jangka Menengah

  • Implementasi Supabase Auth untuk login dashboard web
  • Tidak lagi bergantung pada publicId saja sebagai akses
  • Persiapan untuk fitur multi-akun

4.2 Fitur Dual Akun (Pasangan)

  • Dua nomor WA terhubung ke satu pool data keuangan bersama
  • Notifikasi ke keduanya saat salah satu mencatat transaksi
  • Dashboard bersama dengan breakdown per anggota
  • Opsi "transaksi pribadi" yang tidak terlihat pasangan

4.3 Billing & Subscription Otomatis

  • Integrasi payment gateway (Xendit atau Midtrans)
  • Manajemen tier otomatis: user di-upgrade/downgrade sesuai status pembayaran
  • Notifikasi jatuh tempo dan perpanjangan via WhatsApp
